## Run Locally

Clone the project

```bash
  git clone  https://github.com/farfromsouls/SibSAU-schedule.git
```

Go to the project directory

```bash
  cd SibSAU-schedule
```

Install dependencies

```bash
  pip install beautifulsoup4 lxml aiogram requests
```

Start the server

```bash
  python main.py
```
